# UCI HAR Dataset Tidy Data

This codebook describes the output of run_analysis.R.

The script cleans the [UCI Human Activity Recognition dataset](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones) [1].

## Identifiers
- **subject**: The numeric ID of the test subject.
- **activity**: Which activity the subject performed when the measurements were taken

## Activity Labels
The `activity` column represents the activity the subject was performing when the measurement was taken. Six activity types were measured:

- standing
- sitting
- laying
- walking
- walking_downstairs
- walking_upstairs
